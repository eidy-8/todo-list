# Todo List Project

This project is a simple Todo List application where users can create and manage their tasks efficiently. The application allows users to add, view, update, and delete tasks with various attributes such as title, description, due date, and priority.

## Features

- Create a new project with a title, description, due date, and priority.
- Add tasks to projects and mark them as completed.
- View details of each project and its tasks.
- Update project details and tasks.
- Delete projects and clear completed tasks.
